JTS Project

Описание:
Данный эмулятор в большинстве своем написан с нуля, используются только некоторые участки кода из предыдущих эмуляторов.
При написании эмулятора были задействованы такие средства как Netty, Hibernate, нововведения Java 7.
В дальнейшем, планируется адаптировать эмулятор под структуру датапака PTS серверов
(отсюда и название JTS - Java Test Server, посколько эмулятор всегда будет отставаться тестовым сервером,
и будет отставать по своей реализации от оффициальных серверов, Lineage 2).
Для загрузки PTS датапака применен механизм аннотаций. Документация по ним ещё не составлена,
но когда нибудь будет. До этой поры использовать их можно на свой страх и риск, пользуясь уже готовыми примерами.

Для сборки проекта необходим maven 2, JDK 1.7,
Для компиляции native части в ОС Windows необходимо настроить секцию properties файла pom.xml, находящегося в папке с этим Readme
Описание параметров:
<winCompilerDirectory> - путь до GCC компилятора (MinGW)
<winBisonFlexDir> - путь до анализаторов Flex\Bison

Для работы эмулятора необходима JDK версии 1.7, ос Windows, Linux
